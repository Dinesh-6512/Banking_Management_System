# Banking Management System

A Banking Management System developed using FastAPI, PostgreSQL, SQLAlchemy, and Streamlit.

## Features

- View all bank accounts
- Create a new account
- Update account details
- Delete an account
- REST API with FastAPI
- PostgreSQL database integration
- Streamlit frontend

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic
- Streamlit

## Installation

pip install -r requirements.txt

## Run Backend

uvicorn main:app --reload

## Run Frontend

streamlit run Frontend/app.py
